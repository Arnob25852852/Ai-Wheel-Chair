What is the Ai Wheel Chair?

Introducing the AI Wheel Chair, a new innovation that combines artificial intelligence with mobility assistance. It is designed to improve independence and accessibility for people with mobility challenges — empowering users to engage more confidently with their world.

What do we want to achieve?

Our project aims to deliver a smart AI wheelchair that provides users with full independence: autonomous navigation, continuous health monitoring, and automatic emergency response — improving daily life for people with disabilities and patients.

What science explains the project?

The system integrates several engineering disciplines: DC motors drive motion, GSM/SIM modules enable remote alerts, robotics and control theory provide mechanical intelligence, and modern navigation systems (LiDAR, IMU, computer vision) enable autonomous and assisted mobility.

Disabled people

Impact on Disabled Individuals

AI wheelchairs empower disabled individuals by improving mobility, navigating obstacles like stairs and uneven ground, and increasing access to different environments. With advanced sensors and AI algorithms, these wheelchairs offer personalized help based on the user's needs. This support promotes greater independence and freedom. It allows them to work with others without feeling different.

Impact on Elderly people

AI wheelchairs are not just made for people with disabilities; they also help elderly individuals who face mobility issues due to aging. Movement problems often lead to the retirement of many older adults. However, if we tackle these mobility challenges, they might continue working for more years. In terms of mental tasks, older people could be a better choice. They are more reliable than those they would replace. They know how to do the job well. Therefore, fixing their mobility issues can expand the workforce of any organization. We can address this problem with a wheelchair that has an automated movement system, monitoring software, a live health check-up system, and an easy-to-use control panel, which is what our AI wheelchair offers.

Ai legs

Ai Legs

Ai Legs are a novel feature that enables the wheelchair to traverse challenging terrain or use leg-like actuators when wheels can’t. They can help the device climb small steps or assist in freeing the chair when it gets stuck — acting as a temporary limb or manipulator when needed.

How is the project useful in real life, especially in Bangladesh?

Our AI-powered wheelchair improves mobility, safety, and convenience through continuous monitoring and automated movement assistance — critical in contexts where caregivers may be unavailable. Use cases include Hospitals, Homes, and general support for disabled or elderly people in daily life and work.

Impact: For Bangladesh, the chair can provide independence to many families, improve safety for elderly relatives, and support workforce participation for older adults.



Source Code

Here is the Arduino source code for the AI Wheelchair project:


#include <SoftwareSerial.h>
#include <Wire.h>
#include <LiquidCrystal_I2C.h>

// Pins for GSM
SoftwareSerial gsm(7, 8); 

// Pins for Motors
const int motorRear = 5;
const int motorFrontSteer = 9;
const int emergencyButton = 2;

// Variables
int bpm = 0;
bool emergencyActive = false;

// Initialize Display (Address 0x27 is common)
LiquidCrystal_I2C lcd(0x27, 16, 2);

void setup() {
  pinMode(motorRear, OUTPUT);
  pinMode(motorFrontSteer, OUTPUT);
  pinMode(emergencyButton, INPUT_PULLUP);
  
  Serial.begin(9600);
  gsm.begin(9600);
  lcd.init();
  lcd.backlight();
  
  lcd.setCursor(0, 0);
  lcd.print("AI Wheelchair");
  delay(2000);
}

void loop() {
  readHeartRate();
  updateDisplay();
  
  // Check for Emergency Button
  if (digitalRead(emergencyButton) == LOW) {
    sendEmergencySMS();
  }

  // Basic movement logic (Example: Move forward)
  digitalWrite(motorRear, HIGH); 
  
  delay(500); // Small delay for stability
}

void readHeartRate() {
  int rawValue = analogRead(A0);
  // Simple mapping for demonstration; actual BPM logic 
  // requires pulse timing math
  bpm = map(rawValue, 0, 1023, 40, 160); 
}

void updateDisplay() {
  lcd.setCursor(0, 1);
  lcd.print("BPM: ");
  lcd.print(bpm);
  lcd.print("   "); // Clear trailing digits
}

void sendEmergencySMS() {
  lcd.clear();
  lcd.print("Sending SMS...");
  
  gsm.println("AT+CMGF=1"); // Set SMS mode
  delay(1000);
  gsm.println("AT+CMGS=\"+880 1******878\""); // number
  delay(1000);
  gsm.print("EMERGENCY! The patient needs help. Heart Rate: ");
  gsm.print(bpm);
  gsm.write(26); // CTRL+Z to send
  
  delay(5000); // Prevent spamming
  lcd.clear();
}
